# 📦 FedEx Supply Chain Delivery History - Exploratory Data Analysis (EDA)

## 📖 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the **FedEx Supply Chain Delivery History** dataset. The objective is to understand shipment patterns, delivery performance, freight costs, vendor efficiency, and logistics operations through data cleaning, visualization, and business insights.

The project demonstrates how Python can be used to analyze real-world supply chain data and generate meaningful insights for better decision-making.

---

## 🎯 Objectives

- Clean and preprocess the dataset
- Handle missing values and inconsistent data
- Detect and analyze outliers
- Perform exploratory data analysis (EDA)
- Visualize relationships between variables
- Analyze delivery performance by shipment mode, vendor, and country
- Generate business recommendations based on insights

---

## 📈 Key Insights

- Most shipments are delivered on time.
- Some shipment modes perform better than others.
- Certain vendors consistently provide better delivery performance.
- Delivery performance varies across countries.
- Freight cost generally increases with shipment weight and shipment value.
- Several numerical variables contain genuine business outliers.
- Strong relationships exist between shipment value, insurance, and freight cost.

---

## 💡 Business Recommendations

- Improve shipment modes with higher late delivery rates.
- Partner with vendors that consistently deliver on time.
- Optimize freight costs by selecting cost-effective transportation methods.
- Focus on countries with higher delivery delays.
- Monitor high-value shipments carefully.
- Use dashboards for continuous logistics performance monitoring.

## 🚀 How to Run the Project

1. Clone this repository


2. Navigate to the project folder


3. Install the required libraries
